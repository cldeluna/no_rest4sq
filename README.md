| ------------------------------------------------ | ------------------------------------------------- |

| [streamlit](https://pypi.org/project/streamlit/) | Web Framework for the User Interface              |
| [requests](https://pypi.org/project/requests/)   | Used to send HTTP requests to the SuzieQ REST API |
